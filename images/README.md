# Folder Screenshot Aplikasi

Masukkan gambar/screenshot aplikasi ke folder ini dengan nama file sesuai yang ada di proposal:

## Daftar Gambar yang Diperlukan:

1. **login.png** - Screenshot halaman login
2. **dashboard.png** - Screenshot halaman dashboard
3. **buku-tanah.png** - Screenshot halaman data buku tanah
4. **peminjaman.png** - Screenshot form peminjaman
5. **pengembalian.png** - Screenshot halaman pengembalian
6. **petugas.png** - Screenshot manajemen petugas
7. **laporan.png** - Screenshot halaman laporan

## Tips Screenshot:

- Gunakan resolusi minimal 1366x768
- Format: PNG atau JPG
- Pastikan UI terlihat jelas dan profesional
- Crop area yang tidak perlu
- Gunakan mode fullscreen untuk hasil terbaik

## Cara Mengambil Screenshot:

### Windows:

- **Windows + Shift + S** - Snipping Tool
- **Windows + PrintScreen** - Full screen
- **Alt + PrintScreen** - Active window

### Tools Rekomendasi:

- Greenshot (gratis)
- ShareX (gratis)
- Lightshot (gratis)
